Atoms

Atom are the building blocks of your app, used for layout and rendering of single elements

Button

The Button is used for interactions dealing with revealing modals and routing to another page

{% include styleguide/atoms/Button.html %}

Container

The Container is used to wrap your content so it doesn't bleed to the edge of the page

{% include styleguide/atoms/Container.html %}

SearchInput

The SearchInput is used for `input[type="search"]` elements but are styled specifically for the app

{% include styleguide/atoms/SearchInput.html %}

WrapContainer

The WrapContainer atom is used to wrap content to a certain width
