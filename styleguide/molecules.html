Molecules

Molecules are the building blocks of your app, used for layout and rendering of multiple elements

AppContainer

The AppContainer molecule is used to properly contain the sidebar and the content for the app.

{% include styleguide/molecules/AppContainer.html %}

BareList

The BareList molecule is used to render a list with no default styling

{% include styleguide/molecules/BareList.html %}

InlineList

The InlineList molecule is used to render a list of inline elements

{% include styleguide/molecules/InlineList.html %}

Media

The Media molecule is used to render media such as an image or video, and content together on one line

{% include styleguide/molecules/Media.html %}

Reveal

The Reveal molecule is used to reveal content when an element is clicked

{% include styleguide/molecules/Reveal.html %}

Slideshow

The Slideshow molecule is used to display a collection of images

{% include styleguide/molecules/Slideshow.html %}

Switch

The Switch molecule is just a fancy checkbox with a thumb to switch from off to on
